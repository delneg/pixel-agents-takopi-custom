## Step 7: Permission timer (Python async) — DONE

Implemented in `server/timer_manager.py`:
- `TimerManager` class with `start_permission_timer(agent_id, tg, session)` — launches a 7s sleep in the task group
- On timeout, checks both parent-level and sub-agent tools for non-exempt ones
- Emits `agentToolPermission` for the parent + `subagentToolPermission` for each stuck sub-agent Task
- `cancel_permission_timer(agent_id)` — cancels via `anyio.CancelScope`

Sub-agent tracking in `AgentSession`:
- `active_tool_names: dict[str, str]` — parent tool_id → tool_name
- `active_subagent_tool_names: dict[str, dict[str, str]]` — parentToolId → {subToolId → toolName}
- `clear_activity()` method resets all tracking state

`dangerously_skip_permissions` now configurable via `PIXEL_AGENTS_SKIP_PERMISSIONS` env var (defaults to `"1"` = skip, set to `"0"` to require permissions).

**Limitation:** takopi does not currently translate `bash_progress` / `mcp_progress` events (`StreamEventMessage` is dropped silently), so the permission timer cannot be restarted when a long-running Bash command produces output. This differs from the extension which restarts the timer on bash/mcp progress.

## How to Run

```sh
# 1. Build webview
cd webview-ui && npm install && npm run build && cd ..

# 2. Install server (needs Python 3.14 for takopi)
uv venv --python 3.14 && uv pip install -e server

# 3. Run (from project root)
uv run uvicorn server.app:app --host localhost --port 8765

# 4. Open http://localhost:8765

# 5. (Optional) Run with Telegram bridge
TELEGRAM_BOT_TOKEN=... TELEGRAM_CHAT_ID=... uv run uvicorn server.app:app --host localhost --port 8765
```

## Known Issues / Open Items

1. ~~**No furniture assets**~~ — **RESOLVED.** New `default-layout.json` uses only the 8 hardcoded furniture types (desk, bookshelf, plant, cooler, whiteboard, chair, pc, lamp). `floors.png` generated with 7 grayscale patterns. Generator scripts: `scripts/generate-floors.py`, `scripts/generate-default-layout.py`. If the tileset pipeline is run later, the dynamic catalog overrides these built-in sprites.
2. **bash/mcp progress events** — takopi drops `StreamEventMessage` silently, so the permission timer can't restart on long-running Bash output (unlike the extension).
3. **CWD** — Server uses the directory you run `uvicorn` from as Claude's working directory.
4. **Server must be launched outside Claude Code** — `CLAUDECODE` env var blocks nested sessions. Run the server in a regular terminal.
5. **`PIXEL_AGENTS_SKIP_PERMISSIONS`** — Defaults to `"1"` (skip permissions). Set to `"0"` to enable permission prompts.
