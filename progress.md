## Step 2: Agent manager using takopi's ClaudeRunner ‚Äî DONE

Implemented in `server/agent_manager.py`:
- Uses `ClaudeRunner(dangerously_skip_permissions=True)` from `takopi.runners.claude`
- `set_run_base_dir(self._cwd)` / `reset_run_base_dir(token)` for CWD
- Iterates `runner.run(prompt, session.resume_token)` async iterator
- Updates `session.resume_token` from `StartedEvent`
- Maps each event via `map_event()` and broadcasts to all WebSocket clients
- Tracks `active_tools` per session
- Handles `CompletedEvent` ‚Üí marks `is_running = False`
- Error handling with fallback `agentToolsClear` + `agentStatus: 'waiting'`

**Open issue:** `dangerously_skip_permissions=True` is hardcoded. Should be configurable via env var or config file.

## Step 3: Event mapper ‚Äî TakopiEvent ‚Üí pixel-agents messages ‚Äî DONE

Implemented in `server/event_mapper.py`:
- `StartedEvent` ‚Üí `agentStatus: 'active'`
- `ActionEvent(phase="started")` ‚Üí `agentToolStart` or `subagentToolStart` (if `parent_tool_use_id` present)
- `ActionEvent(phase="started", kind="subagent")` ‚Üí prepends "Subtask:" to status
- `ActionEvent(phase="completed")` ‚Üí `agentToolDone` / `subagentToolDone` / `subagentClear`
- `CompletedEvent` ‚Üí `agentToolsClear` + `agentStatus: 'waiting'` + `agentAnswer` (with answer text)
- `needs_permission_timer()` / `cancels_permission_timer()` helper functions
- Tool name for animation lookup comes from `event.action.detail["name"]` ‚Äî maps to existing `STATUS_TO_TOOL` in webview

**Not yet verified:** Sub-agent event flow hasn't been tested end-to-end with a real Task tool invocation.

## Step 7: Permission timer (Python async) ‚Äî PARTIAL

Implemented in `server/timer_manager.py`:
- `TimerManager` class with `start_permission_timer(agent_id, tg)` ‚Äî launches a 7s sleep in the task group, broadcasts `agentToolPermission` on timeout
- `cancel_permission_timer(agent_id)` ‚Äî cancels via `anyio.CancelScope`
- Integrated in `agent_manager.py`: `needs_permission_timer()` triggers start, `cancels_permission_timer()` cancels on any data

**Not yet verified:** Permission timer has not been tested with a real tool that hangs waiting for permission. The `dangerously_skip_permissions=True` flag means tools won't actually ask for permission, so the timer would only fire if Claude is genuinely slow.

**Missing:** Sub-agent permission detection (`subagentToolPermission` message). The event mapper emits `subagentToolStart`/`subagentToolDone` but doesn't yet track which sub-tools are active for the exempt check (mirrors `activeSubagentToolNames` from the extension).

## How to Run

```sh
# 1. Build webview
cd webview-ui && npm install && npm run build && cd ..

# 2. Install server (needs Python 3.14 for takopi)
uv venv --python 3.14 && uv pip install -e server

# 3. Run (from project root)
uv run uvicorn server.app:app --host localhost --port 8765

# 4. Open http://localhost:8765
```

6. **Prompt sending not tested** ‚Äî Requires `claude` CLI available on PATH. The WebSocket plumbing and ClaudeRunner integration are in place but untested end-to-end with a real Claude process.
